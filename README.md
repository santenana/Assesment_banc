🏦 Sistema de Predicción Bancaria
Este proyecto es una herramienta interactiva desarrollada con Streamlit y PyTorch para la clasificación de riesgos y análisis de datos financieros. Permite procesar datos tabulares, aplicar modelos de Deep Learning (Autoencoder y CNN-LSTM) y visualizar resultados mediante métricas de evaluación y análisis exploratorio.

📋 Prerrequisitos
Antes de comenzar, asegúrate de tener instalado:
* Git

* Python 3.10+ o Conda

* Docker (opcional, para ejecución en contenedores)

git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio
